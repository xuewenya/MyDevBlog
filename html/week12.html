In the Week 12 lesson, I learn about RestDB, it's a simple, secure NoSQL online database backend for web applications. It provide rapid low code development and easy collaboration and data management with secured API-keys for web queries.

After this lesson I know how to use RestDB to create collection and field.

I also learn how to GET and POSE data from RestDB using APIkeys.
